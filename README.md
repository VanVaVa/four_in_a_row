# Игра "Четыре в ряд"

Проект представляюет собой реализацию классической игры "четыре в ряд". Помимо основного функционала игры имеется система локальных пользовательских аккаунтов, хранящихся в localStorage, а также локальная таблица лидеров.

## Используемы подходы

В качестве основного архитектурного подхода был использован Feature Sliced Development. Вся логика инкапсулирована внутри Zustand.Также для стилизации был использован модульный css.

## Как запустить

Для демонстрации можно использовать dev сервер. Для этого нужно выполнить соманду

```bash
npm i
```

А затем запустить сам сервер командой

```bash
npm run dev
```
